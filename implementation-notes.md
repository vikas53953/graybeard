FIXED marketplace.json → `"source": "./"`. This bug would have hit EVERY user, not just this machine. The claude-code-guide agent twice gave wrong values ("local" source, "." plugin source) — the on-disk working examples were the reliable source of truth. Also de-numbered plugin.json + marketplace.json descriptions ("10-minute" → "short") per the never-ship-unmeasured-numbers rule. Canonical registries for reference: ~/.claude/plugins/known_marketplaces.json (marketplaces) + installed_plugins.json (installed plugins, cache paths).
