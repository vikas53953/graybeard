Install it from this folder

No GitHub, no commit needed — it runs straight from disk. Type these three, one at a time.

/plugin marketplace add "C:\Users\vikasmit\Downloads\vikas work\graybeard"
/plugin install graybeard@graybeard
/reload-plugins
Expect: it confirms the marketplace was added and the plugin installed — in well under 2 minutes. If it errors on the path, check the quotes are exactly as above. (Windows: if it says "git not on PATH", that's a known Claude Code quirk — tell me and I'll give you the workaround.)
